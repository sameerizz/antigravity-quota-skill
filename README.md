# Antigravity IDE Quota Architecture Skill

An Agent Skill designed for the [skills.sh](https://skills.sh) ecosystem. This skill provides AI agents with the mental models, API secrets, and troubleshooting steps for building quota monitoring extensions for the Antigravity IDE / Codeium ecosystem.

## Installation

To install this skill into your agent environment, run:

```bash
npx skills add sameerizz/antigravity-quota-skill
```

## Contents
- `SKILL.md`: The core instructions and API documentation for the AI agent.

## Background
The Antigravity IDE "Model Quota" UI is a composite of disjointed backends (Google Cloud Code API and Codeium Language Server). This skill teaches AI agents how to properly probe and understand these APIs to avoid common debugging pitfalls when developing extensions for the IDE.
